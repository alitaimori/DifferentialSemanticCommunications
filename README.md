# DifferentialSemanticCommunications
Implementation codes of the IEEE TMLCN paper:
"Natural transports-inspired differential semantic communications: Intelligent transceivers carry just differences as meanings"

DSC is a closed-loop communication architecture, where intelligent parties communicate like humans transport meanings. In the DSC model, only informational differences are coded, which brings bandwidth efficiency.
There is a folder with the following name:
DSC_Codes: This folder contains implementation codes of the proposed DSCs. In this folder, there is a “readme.txt” file that explains its content in more details.  

Note: For benchmarking and ease of comparison by researchers on the same test image data (337 images of common objects), we have also placed the corresponding results for the compared methods of PNG, JPEG, IMAT Compressed Sensing, and Language-oriented Semantic Communications.
The complete datasets and results used in this work are publicly available on Zenodo as follows:
